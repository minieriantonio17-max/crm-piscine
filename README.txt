# CRM Piscine - PWA Vercel Ready

Questa cartella `public/` è pronta per essere caricata su GitHub e deployata su Vercel.

## Caricamento su Vercel
1. Crea un nuovo repository su GitHub
2. Carica questa cartella `public/` nella root del repo
3. Su Vercel: New Project → collega il repo
4. Imposta "Framework Preset" su Other
5. Output directory: `public`
6. Deploy!

Dopo il deploy:
- Apri il link in Chrome (PC) → Installa come app
- Apri il link in Safari (iPhone) → Condividi → Aggiungi a Home
